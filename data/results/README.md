# Results Directory

This directory contains benchmark results organized by version.

## Structure

```text
results/
├── v0.1.0/
│   ├── gpt-4o.json
│   └── summary.json
└── v0.2.0/
    └── ...
```

## File Format

Each model result file contains:

- Benchmark version
- Model configuration
- Task results with scores
- Aggregate score
- Timestamp

Results are committed to the repository for historical tracking.
